# 🚀 Roosevelt DevOps Platform

A simulated DevOps Operations Console built with Flask, Docker, and Kubernetes.

## ✅ Features

- Role-based login (Admin/User)
- Service control (Restart, Scale, Alert)
- Live logs system
- Monitoring UI
- Admin panel (user management simulation)

## 🛠️ Tech Stack

- Python (Flask)
- Docker
- Kubernetes
- JavaScript (Frontend UI)
